
# F3D-Lab
A platform for doing some experiments about 3D things.

## Datasets

* [COMA](https://coma.is.tue.mpg.de/)
* [FAUST](http://faust.is.tue.mpg.de/)
* [DFAUST](https://dfaust.is.tue.mpg.de/)
* [ShapeNet](https://shapenet.org/)
